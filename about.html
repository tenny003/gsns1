서비스 특징

🌏

지역별 맞춤 순위

아시아, 북미, 유럽, 글로벌 4개 권역의 특성에 맞게 게임 인기도를 분석하여 지역별 최적화된 순위를 제공합니다.

📊

데이터 기반 순위

SteamSpy API 동시접속자(CCU) 데이터와 누적 소유자 수, 지역별 인기 가중치를 결합한 알고리즘으로 순위를 산정합니다.

7개 장르 분류

롤플레잉, 슈팅, 전략, 액션, 스포츠·레이싱, 어드벤처, 시뮬레이션 장르별로 세분화된 순위를 확인할 수 있습니다.

🔄

주간 자동 업데이트

매주 월요일 GitHub Actions를 통해 데이터가 자동으로 갱신됩니다. 항상 최신 게임 트렌드를 반영합니다.

🌐

4개국 언어 지원

한국어, English, 中文, 日本語 4개 언어를 지원합니다. 게임 공식 명칭도 각 언어별 현지화된 이름으로 표시됩니다.

🌙

다크/라이트 모드

사용자 환경에 맞는 테마를 선택할 수 있습니다. 브라우저 세션 간에도 선호 설정이 유지됩니다.

순위 산정 방식

GSNS의 게임 순위는 다음과 같은 방식으로 산정됩니다.

  • Steam 게임: SteamSpy API에서 제공하는 동시접속자(CCU) 수와 누적 소유자 수 중앙값을 기반으로 점수를 계산합니다.
  • 비Steam 게임: Riot Games(리그 오브 레전드, 발로란트), Epic Games(포트나이트), Nintendo 등 각사 공개 발표 플레이어 수를 기반으로 추정치를 산출합니다.
  • 지역 가중치: 각 게임의 지역별 인기도를 반영한 가중치를 적용합니다. 예를 들어, 배틀그라운드는 아시아에서, 콜 오브 듀티는 북미에서 더 높은 가중치를 받습니다.
  • 업데이트 주기: 매주 월요일 새벽 자동으로 최신 데이터를 수집·갱신합니다.

순위 데이터는 게임의 절대적인 인기를 나타내는 것이 아니라, 공개 데이터를 기반으로 한 참고 지표임을 안내드립니다. 모바일 전용 게임, 콘솔 전용 게임의 경우 Steam 데이터가 없어 추정치의 오차가 있을 수 있습니다.

대상 사용자

GSNS는 다음과 같은 분들께 유용합니다.

  • 게이머: 현재 어떤 게임이 인기인지 파악하고 새로운 게임을 발견하고 싶은 분
  • 게임 업계 종사자: 글로벌 게임 시장 트렌드를 모니터링하는 개발사·퍼블리셔 담당자
  • 콘텐츠 크리에이터: 어떤 게임 콘텐츠가 인기인지 참고하고 싶은 스트리머·유튜버
  • 연구자·분석가: 게임 산업 데이터를 수집·분석하는 연구자

지역 분류 기준

  • 아시아: 한국, 일본, 중국, 동남아시아 시장을 포함한 아시아 태평양 권역
  • 북미: 미국, 캐나다를 중심으로 한 북아메리카 권역
  • 유럽: 서유럽, 북유럽, 동유럽을 포함한 유럽 전체 권역
  • 글로벌: 전 세계 종합 데이터 기반 글로벌 순위
